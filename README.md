# Testes manuais em E-commerce
Projeto desenvolvido com objetivo de realizar e praticar testes manuais em um ambiente real de e-commerce, documentando cenários de teste, bugs encontrados e o comportamento da aplicação em diferentes navegadores.

---
## Objetivo
Avaliar a execução da funcionalidade de carrinho de compras de um e-commerce, verificando a eficiência das operações de adicionar, remover e atualizar os produtos selecionados em diferentes navegadores.

---
## Ferramentas utilizadas
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Documentação em Markdown
- Git
  
---
## Funcionalidades testadas
- Adicionar itens ao carrinho de compras
- Remover itens do carrinho de compras
- Alterar a quantidade de produtos
- Atualização dos preços
  
---
## Ambiente de testes
**Aplicação testada:** Americanas
**Navegadores:**
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
  
---

## Resultados
| Navegador | Resultado |
|-----------|-----------|
| Google Chrome | Apresentou bugs críticos |
| Mozilla Firefox | Funcionamento esperado |
| Microsoft Edge | Delay na atualização dos preços |

---
## Bugs encontrados
### Bug 01 - Remoção de itens no Google Chrome
**Severidade:** Crítica
**Descrição:**
Não foi possível remover produtos diretamente pelo carrinho, a mesma se deu apenas ao prosseguir para a etapa de pagamento.

---
### Bug 02 - Atualização de preços no Microsoft Edge
**Severidade:** Média
**Descrição:** 
Após remover um produto de carrinho, o valor total apresentou demora na atualização sendo necessário interagir com a página ou recarregá-la.

---

- Identificação e documentação de bugs
- Testes em diferentes navegadores (Cross Browser Testing)
